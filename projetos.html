Landing Page Promocional – GTA V / GTA Online

Landing page promocional inspirada em GTA V e GTA Online, desenvolvida com HTML5, CSS3 e JavaScript. A estrutura semântica facilita a organização e acessibilidade do conteúdo, enquanto o CSS garante um layout moderno, responsivo e com bom contraste visual. O JavaScript adiciona interatividade, como rolagem suave e ações em botões. O projeto simula uma página real de divulgação, com foco em desempenho e compatibilidade entre dispositivos.

Player de Áudio – Dom Casmurro

Interface de um player de áudio simples desenvolvido com HTML, CSS e JavaScript, inspirado em plataformas de streaming. Exibe a capa do livro "Dom Casmurro", título do capítulo e autor, além de botões interativos para reproduzir, pausar e navegar entre faixas. O layout responsivo utiliza gradiente de fundo, centralização de conteúdo com flexbox e ícones estilizados para controle da reprodução.

Automação de Cadastro com Python

Script automatizado desenvolvido com Python, utilizando as bibliotecas PyAutoGUI, time e pandas para preencher e submeter formulários de forma repetitiva em um site. O código lê dados de um arquivo CSV contendo informações de produtos e, por meio de comandos de teclado e mouse simulados, realiza o login em uma plataforma e insere os dados linha por linha. A automação é útil para tarefas repetitivas em sistemas sem API disponível, otimizando tempo e reduzindo erros manuais.
